# 60–90 second demo script (for LS / QA)

**Title:** GrabbyVault Pro — license activation walkthrough

### Shot 1 (0–10s) — Product

- SilenVault store or LS checkout page  
- VO/text: “GrabbyVault Pro is a Windows video downloader. No ads. Sold via Lemon Squeezy.”

### Shot 2 (10–25s) — Purchase

- Complete test checkout (or show test order)  
- Open receipt email → highlight **license key**

### Shot 3 (25–45s) — Install & activate

- Unzip / run GrabbyVault  
- Click **Pro**  
- Paste key → **Activate**  
- Badge shows **PRO**

### Shot 4 (45–65s) — Pro feature

- Paste a public sample video URL (e.g. short Creative Commons / your own upload)  
- Choose 1080p → download starts  
- Optional: show Settings → ffmpeg OK

### Shot 5 (65–85s) — Seat control

- Click **Release seat** or show **Take over this PC** label  
- Text: “Activation limit 1 — one PC at a time, like a single screen.”

### End card

```text
GrabbyVault Pro · SilenVault
store.silenvault.com
Licensing: Lemon Squeezy License API
```

### Do / don’t for verification recordings

| Do | Don’t |
|----|--------|
| Use test mode / coupon | Promise Netflix/DRM downloads |
| Use short public sample videos | Download copyrighted movies as “demo” |
| Show real license key UI (blur key if sharing publicly) | Share production secret API keys (not used in client) |
